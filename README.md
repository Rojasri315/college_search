# College Discovery Platform

A production-grade MVP inspired by Careers360 and Collegedunia, designed to help students discover, compare, and evaluate colleges efficiently.

## Features

### College Listing & Search
- Search colleges by name
- Dynamic filtering
- Responsive college cards

### College Detail Page
- College overview
- Courses offered
- Fees and ratings
- Placement information

### Compare Colleges
- Select and compare multiple colleges
- Side-by-side comparison of:
  - Fees
  - Ratings
  - Location
  - Placement statistics

### Authentication
- User signup and login
- Password hashing using bcrypt
- JWT-based authentication
- Protected routes

### Saved Colleges
- Save favorite colleges
- Personalized saved colleges page

## Tech Stack

### Frontend
- Next.js 15
- TypeScript
- Tailwind CSS

### Backend
- Next.js Route Handlers
- Prisma ORM
- PostgreSQL (Neon)

### Authentication
- JWT
- bcrypt

## Database Models
- User
